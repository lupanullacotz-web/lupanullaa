# Lupanulla Website

Static HTML project for the Lupanulla education portal.

## Structure

- `index.html` - main portal page
- `masomo.html` - subjects
- `mitihani.html` - past papers
- `blogu.html` - news and learning articles
- `kamusi.html` - subject dictionary
- `calculator.html` - score calculator
- `videos.html` - Lupa+ videos
- `ajira.html` - education jobs
- `walimu.html` - teacher resources
- `vyuo.html` - colleges and course guidance
- `mikoa.html` - regional/school rankings
- `matangazo.html` - announcements
- `dashboard.html` - student dashboard
- `admin.html` - static admin dashboard mockup
- `fisimaji.html` - AI assistant page
- `terms.html` and `privacy.html` - legal pages

## Assets

- `assets/css/main.css` contains shared styles and small Tailwind-friendly utility classes.
- `assets/js/main.js` contains shared behavior for component includes, mobile menu toggles, active links, and simple search filters.
- `assets/js/dashboard.js` contains reusable sidebar behavior for dashboard-style pages.
- `components/` contains reusable HTML snippets for future pages.

## Notes

Most source pages use Tailwind, Phosphor Icons, and Google Fonts from CDNs, so an internet connection is needed for the final styling/icons to load fully.
